# 📚 Exercícios JS FaFla

Projeto desenvolvido para praticar JavaScript básico através de 100 exercícios interativos executados diretamente no navegador. Cada exercício trabalha conceitos fundamentais da linguagem, como operações matemáticas, estruturas condicionais, laços de repetição, arrays, objetos e manipulação de strings.

---

## 🎯 Objetivo

Auxiliar estudantes iniciantes a praticarem lógica de programação e JavaScript de forma simples e interativa, utilizando botões para executar cada exercício individualmente.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📋 Funcionalidades

- Interface simples e intuitiva
- 100 exercícios organizados em botões
- Entrada de dados utilizando `prompt()`
- Exibição de resultados com `alert()`
- Exercícios de lógica de programação
- Operações matemáticas
- Estruturas condicionais
- Laços de repetição
- Arrays e objetos
- Manipulação de strings
